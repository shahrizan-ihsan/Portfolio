## PART 2 — AI SESSION FLOW (Step by Step)

### Every Single Session:

**STEP 1** — Open a new chat
- Claude: claude.ai
- ChatGPT: chatgpt.com

**STEP 2** — Paste the Master Context Block (Part 1 above)

**STEP 3** — Add your Task Template (pick from Part 3 below)

**STEP 4** — Dump your Raw Material (everything you have — messy is fine)

**STEP 5** — Send and get your first draft

**STEP 6** — Iterate with short follow-up prompts:
- "Make the hook punchier"
- "Shorten slide 3 to one line"
- "More personal, less formal"
- "Change the CTA to a question"
- "Give me 3 alternative headlines"

**STEP 7** — Copy output to the right place:
- LinkedIn draft → paste and schedule
- Canva → paste carousel text into slides
- GitHub → save to your Week-XX file
- Slack → share with team if needed

**STEP 8** — Save any prompt that worked well to Claude.md on GitHub

## PART 4 — GITHUB UPDATE SOP

### When to Update GitHub
- After publishing each weekly retrospective → add Week-XX file
- When your skills or tools change → update Skills.md
- When you discover a great new AI prompt → add to Claude.md
- When the project evolves → update README.md or About-Shah.md

### How to Edit an Existing File
1. Go to github.com/shahrizanihsan-source/shahs-portfolio
2. Click the file name
3. Click the pencil icon (top right)
4. Make your edits
5. Click "Commit changes..."
6. Write a short commit message: what + why
7. Click "Commit changes" — done

### How to Create a New File
1. Go to your repo main page
2. Click the + button → "Create new file"
3. Name it: Week-01-STAILISHROOM.md
4. Write your content
5. Commit with a clear message

### Commit Message Formula
```
[Action] [File] — [What changed or why]

Examples:
Add Week-01-STAILISHROOM.md — Project retrospective and carousel script
Update Skills.md — Add Week 3 project and new tools
Update Claude.md — Add new brainstorming prompt that worked well
```

### GitHub Keyboard Shortcuts
- Press T → file finder (search any file instantly)
- Press E → edit current file directly
- Press . (period) → opens VS Code in browser (best for editing multiple files)

## PART 7 — WEEKLY RHYTHM

### Every Week (Friday or Sunday prep):
1. Review the project for that week from your Career Asset Vault
2. Open Claude or ChatGPT — paste Master Context Block
3. Use Weekly Retrospective Task Template
4. Iterate until the carousel script feels right
5. Move to Canva — build the slides
6. Publish on LinkedIn (Monday/Tuesday morning is best for SEA)
7. Post IG companion caption within 2 hours of LinkedIn
8. Save the week's work to GitHub (Week-XX file)
9. Update Skills.md if a new competency was demonstrated
10. Add any new prompts to Claude.md
