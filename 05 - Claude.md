## How Shah Works With Claude

### Prompt Style That Works Best
- Be specific about context
- Share the raw story first — facts, feelings, and outcomes
- Iterate in rounds — first draft, refine tone, sharpen headline, finalise CTA
- Always reference voice: personal, reflective, authentic — not corporate

### Shah Voice Guidelines for Claude
- Warm, grounded, confident — not boastful
- Storytelling-first — lead with the human moment
- Regional sensitivity — understands SEA marketing culture
- Minimalist language — punchy, not padded
- Authentic and reflective — this is a career legacy project

Info: [paste your info here]

### GitHub File Creation Prompt

## Best Practices

1. Always give Claude context — the more Shah shares, the better the output
2. Paste raw assets — old reports, bullet points, rough notes work great as input
3. Specify the platform — LinkedIn vs IG vs email all need different tones
4. Iterate, do not accept first draft — ask Claude to make it punchier or more personal
5. Save good prompts — add to the Prompts Library section above
